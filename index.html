核心特性

🎯

智能插入位置

基于 AST 分析，精确计算最佳插入位置。支持函数参数、return 语句、条件表达式等 17+ 种代码上下文。

🎨

彩色输出

自动生成随机背景色，根据亮度智能选择文字颜色。支持自定义颜色列表和字体大小。

😊

Emoji 前缀

随机 emoji 前缀让日志更易识别。支持自定义 emoji 列表，快速定位不同的日志输出。

🔍

上下文感知

自动包含类名、函数名等上下文信息。支持自定义分隔符和上下文显示开关。

🧩

智能扩展

自动扩展属性访问链（code → res.code）。支持对象内部属性路径构建和可选链操作符。

🛠️

批量管理

一键注释、取消注释、删除所有日志。支持多行日志语句和括号匹配检测。

📝

TypeScript 支持

完整支持 TypeScript 语法。自动过滤类型声明、接口、枚举等无意义位置。

⚡

多框架支持

支持 Vue、Svelte、Astro 单文件组件。自动提取 script 部分进行分析。

快速开始

⚙️ 配置选项

insertContext 插入上下文信息（类名和函数名）
makeLogColorful 启用彩色日志输出
randomEmojiPrefix 添加随机 emoji 前缀
logMessageFontSize 日志字体大小（默认 16px）
customEmojis 自定义 emoji 列表
customColors 自定义颜色列表
查看完整配置 →

⌨️ 快捷键

Cmd + Alt + L 插入日志
Cmd + Alt + C 注释所有日志
Cmd + Alt + U 取消注释所有日志
Cmd + Alt + D L 删除所有日志

Windows/Linux 使用 Ctrl 替代 Cmd
